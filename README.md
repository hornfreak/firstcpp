# firstcpp

## Overview
A C++ project focused on simulation engineering. This project utilizes:
* **CMake** for build automation.
* **fmt** library for modern formatting.
* **clangd** for language server protocol support.

## Environment
* **OS:** Omarchy Linux
* **Editor:** Neovim (LazyVim)
* **Compiler:** Clang/GCC

## Building
```bash
mkdir build && cd build
cmake ..
make
```
